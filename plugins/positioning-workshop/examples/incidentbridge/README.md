# IncidentBridge Example

This is a fictional example that shows the plugin's intended quality bar end to end.

IncidentBridge is an imaginary B2B SaaS product for incident investigation. The example is designed to show how the workshop moves from raw context to a final messaging pack and then into controlled variants.

- `04b-team-synthesis.md`
- `05-messaging-pack.md`
- `06-variant-matrix.md`

## Notes

- Company, metrics, and market details are illustrative.
- The point is to demonstrate structure, tone, and decision quality.
- The example stays grounded in believable early-stage SaaS constraints.
